# Learning_Python_Day18

Today's topic is the requests library!

**Goal:** Learn to make HTTP requests and interact with web resources using Python

- **task1:** Make a GET request to a public API (`https://api.github.com`)

- **task2:** Check response status codes

- **task3:** Access response headers and response body

- **task4:** Pass query parameters in requests

- **task5:** Handle connection errors gracefully

**mini-project:**
Simple Web Data Fetcher:

- Fetch data from the JSONPlaceholder API (`https://jsonplaceholder.typicode.com/posts`)  
- Display the response status code and headers  
- Extract and print specific fields from the response data  
- Implement error handling for network issues  
- Add user input to choose different API endpoints
